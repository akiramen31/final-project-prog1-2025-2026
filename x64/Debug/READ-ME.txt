1. PROJECT OVERVIEW
-------------------
This project is a C-based clone of the classic game "Bomberman," 
developed using the CSFML. The game features a playable character, 
destructible environments, enemies, and a power-up system 
.

2. CONTROLS
-----------
- Movement: 
    - Z: Up
    - S: Down
    - Q: Left
    - D: Right
- Action:
    - Space: Plant Bomb
- System:
    - Escape: Return to Menu

3. GAME MECHANICS
-----------------
- Goal: Defeat all enemies on the map to make the exit accessible. 
  Touching an enemy or being caught in a bomb blast will result 
  in losing a life.
- Bombs: When planted, bombs explode after a short delay, 
  releasing deflagration (fire) in four directions.
- Destructible Boxes: Fire can destroy boxes, which may reveal 
  power-ups or the exit.
- Power-Ups: Collected items can increase player Speed, 
  Bomb capacity, or Fire (explosion radius).
- Invincibility: After losing a life, the player is temporarily 
  invincible for 3 seconds.

1. PRÉSENTATION DU PROJET
-------------------
Ce projet est un clone en C du jeu classique « Bomberman », 
développé à l'aide du CSFML. Le jeu propose un personnage jouable, 
des environnements destructibles, des ennemis et un système de bonus 
.

2. COMMANDES
--------- --
- Déplacement : 
    - Z : Haut
    - S : Bas
    - Q : Gauche
    - D : Droite
- Action :
    - Espace : Placer une bombe
- Système :
    - Échap : Retour au menu

3. MÉCANIQUE DU JEU
-----------------
- Objectif : Éliminer tous les ennemis sur la carte pour rendre la sortie accessible. 
  Toucher un ennemi ou être pris dans l'explosion d'une bombe entraîne 
  la perte d'une vie.
- Bombes : Une fois posées, les bombes explosent après un court délai, 
  provoquant une déflagration (feu) dans quatre directions.
- Caisses destructibles : Le feu peut détruire les caisses, ce qui peut révéler 
  des bonus ou la sortie.
- Bonus : les objets ramassés peuvent augmenter la vitesse du joueur, 
  sa capacité en bombes ou la puissance de ses tirs (rayon d'explosion).
- Invincibilité : après avoir perdu une vie, le joueur est temporairement 
  invincible pendant 3 secondes.
